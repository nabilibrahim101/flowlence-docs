# Download & Install

## System Requirements

- **Operating System:** Windows 10 or later
- **RAM:** 4 GB minimum
- **Disk Space:** 2 GB free space
- **USB Port:** Required for connecting the ESP32

## Download

Download the latest version of Flowlence Code from the official website:

[:material-download: Download Flowlence Code](https://flowlence.com/code-cloud.html){ .md-button .md-button--primary }

## Installation

### Step 1: Run the Installer

Double-click the downloaded `.exe` file to start the installation.

![Installer Welcome](../images/placeholder-install-1.png)

### Step 2: Choose Installation Location

The default location is `C:\Program Files\Flowlence Code`. Click **Install** to proceed.

### Step 3: Wait for Installation

The installer will download required components. This may take a few minutes depending on your internet connection.

### Step 4: Launch Flowlence Code

Once installation is complete, click **Finish** to launch Flowlence Code.

![Flowlence Code Running](../images/placeholder-install-2.png)

!!! success "You're ready!"
    If you see the Flowlence Code interface, the installation was successful. Continue to the [Interface Tour](interface.md) to learn your way around.

## Troubleshooting

!!! warning "Installation stuck at 17%?"
    This usually means the external resources are still downloading. Make sure you have a stable internet connection and wait for the download to complete.

!!! warning "Windows SmartScreen warning?"
    If Windows shows a SmartScreen warning, click **More info** and then **Run anyway**. This happens because the app is new and hasn't yet built up a reputation with Microsoft.
